#  React --  An Online Library Management System Application

Welcome to the Online Library Management System! This application is a fully functional   React web application built using Vite, React Router for navigation, Redux Toolkit for state management, and customized with Vanilla CSS. 

Users can browse books by specific categories, search for titles or authors, view detailed information about each book, and dynamically append new books to the catalog via a validated entry form and also may delete the book.

 # Prerequisites : 
 
 -- Before launching the workspace locally, ensure you have the following installed on your machine:
 
 # Node.js (v16.x or higher recommended)
 
 # npm (Node Package Manager)
 
 
 
 
 
 # How to Run the ApplicationFollow these simple steps to get your local server running: 
 
1. Extract and Navigate to the DirectoryUnzip the project file or clone your repository, open your terminal application, and move into the project root:

# Terminal : cd online-library

2. Install Project Dependencies Run the installation command to download the required node modules package configurations (including React Router and Redux Toolkit):

# Terminal : npm install


3. Start the Development ServerLaunch the local Vite build asset watcher engine by typing:

# Terminal: npm run dev

4. Open the Web BrowserOnce the server spins up, look at the terminal logs and click or browse to the provided local URL address interface (usually http://localhost:5173).
